## Part 2: Vim
**Normal Mode**
* to move the cursor: 
    k to move up
    j to move down
    h to move left
    j to move right

* to replace letter:
    rx to replace the letter with x

> in this part:
> use **rC** to replace character V to C
> use **r0** to replace 1 to 0
